# Employee-Attrition-Analysis
This project contains an analysis of an HR dataset that determines what factors are most associated with employee attrition using Microsoft Excel and Microsoft Power BI.

1) Overview:
The goal of this analysis project is to determine which factors influence employee attrition the most, for the purpose of keeping employees for the longest, and
reducing the costs and efforts of hiring and training of the new employees. 

2) Data Description: 
Source:” Employee Attrition” - Prashant Patel on Kaggle https://www.kaggle.com/datasets/patelprashant/employee-attrition 
Rows: 1470
Used Fields: Attrition, Department, Employee Number, Environment Satisfaction, Job Satisfaction, Overtime, Relationship Satisfaction.

4) Tools Used:
Microsoft Excel: Data cleaning
Microsoft Power BI: trend identification, data visualization and DAX measures

Checked for any duplicate rows, spelling errors or inconsistant formats
